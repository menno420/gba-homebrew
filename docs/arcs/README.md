# Game arcs — design docs

> **Status:** `living-ledger` — index of the multi-slice game arcs' design
> docs. One file per arc; each is the design of record for its game, updated
> as slices land (source code always wins). A reachability root for the
> substrate doc-hygiene gate — new arc docs are linked from here.

An **arc** is a multi-slice game concept driven from a design doc to a v1.0,
each slice landing on green with a headless proof (the Gloamline precedent,
PRs #50–#75). Older concept docs live under [`../concepts/`](../concepts/);
this directory holds the newer, slice-planned arcs.
