## Stability baseline

- **GBA toolchain** (sessions 1–4, re-verified every session since):
  pinned devkitARM r68 + Butano 21.7.1 via `tools/setup-toolchain.sh`
  (idempotent, SHA-256-pinned packages since PR #13); one-command
  recipe in [`BUILDING.md`](BUILDING.md). **NDS toolchain**: pinned
  BlocksDS 1.21.1 + Wonderful GCC via `tools/setup-nds-toolchain.sh`.
  "ROM builds" CI rebuilds every `games/*/Makefile` ROM from source on
  each PR and logs sha256s, so the committed `dist/` binaries are
  always cross-checkable; all shipped ROMs are byte-deterministic
  (two clean builds, identical hash — stated per row in
  [`dist/README.md`](../dist/README.md)).
- **Headless proof loop**: `tools/headless-screenshot.py` (mGBA) and
  `tools/nds-headless-check.py` (DeSmuME) drive scripted inputs and
  assert text/pixels/ELF-resolved memory watches against per-game
  telemetry mailboxes; routes are recorded (`tools/route-recorder.py`,
  `tools/gloam-route.py`, `tools/brine-save.py` fixtures) and replayed
  deterministically. **Committed-proofs convention: every GBA game's
  proofs live in the repo** — `games/<game>/proofs.sh` for
  Cindervault, Deepcast, Clockwork Courier, Shoal and Wickroad (born
  with the game or ported in #112/#113; the Shoal oracle lesson), and
  Lumen Drift's replay tiers as pinned steps in `headless-boot.yml`.
  The NDS games mirror their whole sims host-side
  (`tools/check-gloam.py`, `tools/check-brine.py`) with committed
  routes, proven in the `nds-rom-build` CI job.
- **Web/mobile proof loop**: pure-Node engine smokes plus real-Chromium
  browser smokes (`tools/web-smoke-undertow.mjs`,
  `tools/web-smoke-tiltstone.mjs`, `tools/mobile-foundation-smoke/`),
  run-twice byte-identity as the standard evidence bar.
- **Release packaging**: `tools/package-web-arcade.sh` rebuilds
  `dist/web/` (host-ready static arcade) + `dist/releases/`
  (deterministic versioned zips) and `--verify` asserts every sha256
  pin in [`RELEASES.md`](RELEASES.md) — machinery from PR #109,
  repinned at HEAD by the 2026-07-14 arcade-refresh slice.
